Instructions de déploiement de l'application web:

Note : Veuillez la liste des éléments requis pour le lancement de l'application

1) Compilation du fichier WAR :
 a) Ouvrir une fenêtre de commande
 c) Accéder à la racine de votre projet sous l'invite de commande
 d) Lancer la commande mvn package
 e) Le WAR se trouvera sous le répertoire 'target' de votre projet

2) Déploiement sous Tomcat
 a) Copier votre fichier WAR
 b) Accéder au répertoire de votre serveur tomcat
 c) Déposer le WAR sous répertoire webapp (Il est recommandé de renommer votre WAR pour un nom plus convivial.)
 d) Au démarrage du serveur, le fichier WAR sera automatiquement déployé.

//TODO: TOMCAT
3) Environnement de travail
 a) Si vous n'avez pas de serveur tomcat préinstallé, télécharger la version la plus récente sur le site de tomcat (http://tomcat.apache.org/)
 b) Si vous disposez d'une installation Tomcat dans un dossier ou vous n'avez pas l'accès en écriture, créez un dossier tomcat dans un endroit ou vous les possédez, et créez les dossiers suivants:
  b.1) conf, lib, logs, temp, webapps, work
 b) Configurer les variables d'environnement suivantes:
  a.1) CATALINA_HOME : Répertoire racine de tomcat, c'est à dire le dossier parent du dossier bin.
  a.2) CATALINA_BASE : Répertoire de vos dossiers Tomcat personnels, si existant (Par exemple : "J:/tomcat"), identique à CATALINA_HOME si vous employez directement Tomcat

4) Démarrage de Tomcat
 a) Accéder au répertoire "bin" sous le répertoire racine de votre serveur tomcat
 b) Lancer startup.bat

5) Accéder à l'application
 a) Démarrer votre navigateur web (nous recommandons Firefox ou Google Chrome)
 b) Accéder à : http://localhost:8080/NOM_DE_VOTRE_WAR (sans extension ".war")
